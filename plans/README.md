# Historical implementation plans

This directory contains dated implementation plans, TDD notes, and design exercises from earlier phases of the project.

## Status

These plans are **historical planning artifacts**. They are useful for understanding intent and sequencing at the time they were written, but they may not match the repository's current architecture or completion state.

## Use this directory for

- reconstructing why a feature was approached a certain way
- finding old acceptance criteria or phase breakdowns
- understanding the context behind older commits and bd issues

## Do not use this directory as the current source of truth

For current truth, prefer:
- `openspec/changes/` for active proposals
- `openspec/specs/` for implemented OpenSpec-backed behavior
- `bd` issues for live work tracking
- code/tests/docs for actual current behavior

## Maintenance guideline

If a plan is superseded, leave it here as a dated artifact unless there is a strong reason to rewrite history. Add new planning work in OpenSpec and bd rather than mutating old plans to look current.
